CSS can be added to HTML documents in 3 ways:

  • Inline - byg usin the style attribute inside HTML elementsl
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS files. However,in this tutorial we will use inline and internal styles,because this is easier to demonstrate, and easier for you to try it yourself

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements. ID,on the other heand, applies a style to one unique element. ID is also special in that you can use a special URL to link dirctly to an element and it's used by javaScript.

In CSS, slectors are used to target a specific element or range of element on a web page. Once an elment has been targetd, a style or set of style can be applied to the element.

There is a wide range of selector available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
